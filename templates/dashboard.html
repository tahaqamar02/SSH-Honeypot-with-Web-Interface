Welcome to the Admin Dashboard of a demo SSH server honeypot.

This serves as a simple web interface presented to a potential hacker. The login credentials for this page differ from those used by administrators and users on the real server. The website and admin interface act as a decoy for the hacker, displaying deceptive (fake) data (represented by these files). The intention is to mislead the attacker into believing that the data holds value for the server and site. However, the information is entirely fabricated and mimics the format of the data found on the real server or site. This is the simplest implementation of a Honeypot on a web application.
